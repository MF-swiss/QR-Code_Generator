QRCode Generator - Portable Version

So startest du das Programm:

1. Entpacke das ZIP-Archiv komplett (inkl. Ordnerstruktur).
2. Stelle sicher, dass der Ordner `javafx-sdk-27` und der Unterordner `target` mit dem JAR vorhanden sind.
3. Doppelklicke auf `run-portable.bat`.

Voraussetzungen:
- Java 17 oder neuer muss installiert und in der PATH-Umgebung verfügbar sein (empfohlen: Java 25).
- Es werden keine Adminrechte benötigt.

Fehler? Dann führe `run-portable.bat` in der Eingabeaufforderung aus und kopiere die Ausgabe für Support.
